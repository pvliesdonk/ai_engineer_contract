# Label taxonomy

Standard labels (from `ai/manifest.json`) used by this contract:

- from-ai
- needs-review
- docs
- chore
- security
- blocked
- planning
- needs-design-ref
- breaking-change
- content
- design
- asset
- deviation-approved

> Use `python tools/ai_contract_cli.py labels init` to create them in your repo.
